# Skripsi Dashboard - React Version

Modern, responsive dashboard for classification system built with React, TypeScript, and Tailwind CSS.

## Features

- **Responsive Design**: Optimized for both mobile and desktop
- **Collapsible Sidebar**: Toggle sidebar on desktop, slide-in menu on mobile
- **Interactive Stats**: Clickable stat cards that navigate to relevant sections
- **Data Tables**: Display teacher data, classification results, and processing history
- **File Upload**: Drag-and-drop file upload with CSV/Excel support
- **Smooth Animations**: Powered by Motion (Framer Motion)

## Tech Stack

- React 18.3.1
- TypeScript
- Tailwind CSS 4.1.12
- Motion (Framer Motion) 12.23.24
- Lucide React (icons)
- Vite 6.3.5

## Getting Started

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm run dev
```

### Build

```bash
pnpm run build
```

## Dashboard Sections

1. **Dashboard** - Overview with statistics and quick start guide
2. **Data Pengajar** - Teacher data management and upload
3. **Rekapan Klasifikasi** - Classification results and scores
4. **Pre-Processing** - Data processing pipeline and history

## Design

Features an academic-editorial aesthetic with:
- Crimson Pro (serif) for headlines
- Outfit (sans-serif) for body text
- Warm color palette: Navy, Cream, Terracotta
- Smooth animations and transitions
